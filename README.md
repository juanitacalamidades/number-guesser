Functions used:

    • generateRandomNumber → It generates a random number between 1 and 100 and returns it.
    
    • getPlayerGuess → An input to get the player's guess, and if not introduced correctly, it shows a message to warn them.
    
    • checkGuess → It compares the player's guess with the number generated by the computer and returns a message saying if the guess it's too low, too high or correct.
    
    • game → It shows the structure messages, creates a 10-round limit, reads the player's guess, checks if it's correct, counts the player's attempts and shows if the player has won or lost.
